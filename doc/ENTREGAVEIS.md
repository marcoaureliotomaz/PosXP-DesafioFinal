# Entregaveis Finais

## Itens Do Desafio Atendidos

- API REST no padrao MVC
- CRUD completo do dominio `Cliente`
- Endpoint para listar todos os registros
- Endpoint para buscar por ID
- Endpoint para buscar por nome
- Endpoint para contagem total de registros
- Persistencia com Spring Data JPA e H2
- Estrutura de pastas organizada por responsabilidade
- Documentacao da API com Swagger / OpenAPI
- Testes automatizados
- Execucao via Docker
- Collection do Postman
